Name is the name of the form element while the Value tells us what type of value is going to be stored in that form element

3.    Explain how secure the HTML form input type password is.

The password type is the baseline level of security. While the type masks the password, it's transferred over the network as plaintext, so it must be combined with other security measures such as HTTPS encryption and password hashing to improve its security.
