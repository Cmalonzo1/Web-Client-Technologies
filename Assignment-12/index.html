    1. Find the number 3 in the code. Try replacing it with the following numbers: 0, 1, 2, 3, 4, 100. What does each number do? Why?

      The number selects the child node inside of the parent node "div1" to insert text before that child. While not exactly the same, this node system serves as like an array of nodes, where the parent node is the array and the children are the items inside of it.

    2. For the value 3, compare having the two paragraphs on separate lines (as they are above) to having them on the same line with no spaces between them.

      When the paragraphs are on separate lines, the whitespace is treated as a child node. When the paragraphs are on the same line, there are two child nodes. With the whitespace, there are three.
